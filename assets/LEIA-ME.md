# assets/

Esta pasta contém recursos visuais e a apresentação final do projeto.

## Ficheiros a incluir

| Ficheiro                     | Descrição                                            |
|------------------------------|------------------------------------------------------|
| `apresentacao.pptx`          | Slides da apresentação final (Sessão 5)              |
| `diagrama_arquitetura.png`   | Diagrama das camadas UI / BLL / DAL / BD             |
| `diagrama_bd.png`            | Diagrama entidade-relação da base de dados (se aplicável) |

## Notas

- Exporta os diagramas em **PNG** ou **PDF** para que sejam visualizáveis
  diretamente no GitHub, sem precisar de abrir o ficheiro original.
- O diagrama de arquitetura pode ser criado no **Lucidchart** e exportado daqui.
- A apresentação em `.pptx` não é visualizável no GitHub, mas fica guardada
  no repositório para entrega.
